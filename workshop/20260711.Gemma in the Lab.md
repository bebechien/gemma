### LM Studio

https://lmstudio.ai/

- Load Gemma 4 E4B (API Identifier - `gemma-4-e4b-it`)
- Set Context Length -> 12800 (하고자 하는 작업에 맞추어 설정)
- Set Max Concurrent Predictions -> 몇개의 병렬 작업을 진행할 것인가
