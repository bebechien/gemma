## PC에서 Gemma 4 실행하기

### ollama

https://ollama.com/

```sh
ollama run gemma4:e2b
```

질문 예시:
```
구미호로 짧은 시를 써줘.
```

### LM Studio

https://lmstudio.ai/

- Load Gemma 4 E4B (API Identifier - `gemma-4-e4b-it`)
- Set Context Length -> 12800 (하고자 하는 작업에 맞추어 설정)
- Set Max Concurrent Predictions -> 몇개의 병렬 작업을 진행할 것인가

### 병렬 작업 데모

에이전트에 의해 여러 작업을 동시에 병렬로 진행 가능

Mac 환경: https://github.com/google-gemma/cookbook/tree/main/apps/concurrent

Windows + LM Studio: https://github.com/bebechien/pararell-agent-web-generator-demo

### Self-Hosted Platform

- 데스크탑에서 간단히 사용: https://anythingllm.com/
- 웹페이지로 여러 사용자가 동시에 사용: https://openwebui.com/
- Open-source ChatGPT replacement: https://jan.ai/
- general-purpose AI agent: https://goose-docs.ai/
- 지속적으로 성장하는 agent: https://github.com/nousresearch/hermes-agent
- AI 코딩 에이전트
  - opencode: https://opencode.ai/
  - Pi: https://pi.dev/
- 파인 튜닝 특화: https://unsloth.ai/

example config for opencode `~/.config/opencode/opencode.jsonc`

```
{
  "$schema": "https://opencode.ai/config.json",

  "provider": {
    "lmstudio": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "LM Studio (local)",
      "options": {
        "baseURL": "http://127.0.0.1:1234/v1"
      },
      "models": {
        "google/gemma-4-12b-qat": { "name": "Gemma 4 12B (local)" },
        "google/gemma-4-31b-qat": { "name": "Gemma 4 31B (local)" }
      }
    }
  },

  "model": "google/gemma-4-12b-qat"
}
```
