### LM Studio

https://lmstudio.ai/

- Load Gemma 4 E4B (API Identifier - `gemma-4-e4b-it`)
- Set Context Length -> 12800 (タスクに応じて調整)
- Set Max Concurrent Predictions ->　何個の並列処理を行うか

### 並列作業のデモ

エージェントにより複数の作業の並列処理が可能

Mac 環境: https://github.com/google-gemma/cookbook/tree/main/apps/concurrent

Windows + LM Studio: https://github.com/bebechien/pararell-agent-web-generator-demo

### Self-Hosted Platform

- デスクトップで簡単に使用： https://anythingllm.com/

- 持続的に進化する agent： https://github.com/nousresearch/hermes-agent
- AIコーディングエージェント
  - opencode： https://opencode.ai/
  - Pi： https://pi.dev/
- ファインチューニング特化： https://unsloth.ai/

example config for opencode `~/.config/opencode/opencode.jsonc`

```
{
  "$schema": "https://opencode.ai/config.json",

  "provider": {
    "lmstudio": {
      "npm": "@ai-sdk/openai-compatible",
      "name": "LM Studio (local)",
      "options": {
        "baseURL": "http://127.0.0.1:1234/v1"
      },
      "models": {
        "google/gemma-4-12b-qat": { "name": "Gemma 4 12B (local)" },
        "google/gemma-4-31b-qat": { "name": "Gemma 4 31B (local)" }
      }
    }
  },

  "model": "google/gemma-4-12b-qat"
}
```
